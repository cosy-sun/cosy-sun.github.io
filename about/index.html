About

Progress was built by Alexander Russell and inspired by Cody House’s Reading Progress Indicator. The folks over at Cody House are pretty fantastic and are making some awesome things. They’re also super responsive to comments, so head on over there and check out what they’ve been doing.

Progress features quick and simple environment is set up so users can input all the necesarry info in the config file and start blogging. Further reiterations are in progress to turn this into a Wordpress theme or make it fully editable with Cloudcannon CMS. I cannot let you resell this unless you contact me

If you’d like give credit somewhere on your blog or tweet a shout out to @alexanderussell, that would be pretty sweet.

If you feel like donating — Give it a thought.
