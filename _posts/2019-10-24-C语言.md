#####结构

- #####对结构的合法操作只有拷贝, 作为一个单元对其赋值,通过&操作符以及访问结构成员这几种

- 结构的基本知识

    ```c
    struct point {
	int x;
	int y;
    };

    struct reat {
	struct point pt1;
	struct point pt2;
    };

    struct reat screen;

    test() {
	screen.pt1.x = 10;
	screen.pt2.x = 11;
    }
    ```
    
- 结构与函数

    ```c
    struct point makePoint(int x, int y) {
	struct point p;
	p.x = x;
	p.y = y;
	return p;
}
    ```
    
    ```c
    pointStruct() {
	struct point *pp, p;
	pp = &p;//pp是指向结构p的指针
	(*pp).x;
	(*pp).y;//*pp是结构p, *pp加括号, .运                                算符比*优先级高
    }
    ```
    
    - p是指向结构的指针,那么p->结构成员可以访问结构成员

            pp->x;
	        pp->y;
	        
    - 结构运算符 . -> 函数调用 () 下表[]优先级最高,结合最紧密
